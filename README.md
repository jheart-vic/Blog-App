# BLOG-APP

> A Blog App that uses to the CRUD methods


## Built With

- `Ruby on Rail`

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- `Computer`
### Setup
- Clone  `https://github.com/jheart-vic/Blog-App.git`
### Install
- `Ruby`
- `Rail`
- `PostgresSql`

### Testing
- Run `rspec/spec/folder_name/file_name`
